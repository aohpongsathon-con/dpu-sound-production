01 · พื้นฐาน

ระบบเสียงดิจิตอล

Analog vs Digital Audio

ADCSamplingQuantization

เสียงในธรรมชาติเป็นคลื่น Analog (ต่อเนื่อง) เมื่อบันทึกผ่านไมโครโฟน จะถูกแปลงเป็นข้อมูลดิจิตอลโดย ADC (Analog-to-Digital Converter) ผ่าน 2 ขั้นตอน

Sampling (สุ่มตัวอย่าง) — วัดค่าคลื่นตามแนวเวลา กี่ครั้ง/วินาที = Sample Rate
Quantization (แปลงเป็นตัวเลข) — วัดระดับแอมพลิจูด ความละเอียด = Bit Depth

ยิ่ง Sample Rate สูงและ Bit Depth มาก คุณภาพยิ่งใกล้เคียง Analog ต้นฉบับ แต่ขนาดไฟล์จะใหญ่ขึ้นตามสัดส่วน

ทำไม Sample = แนวตั้ง, Bit Depth = แนวนอน?

ลองนึกภาพกริด: Sample Rate คือจำนวนแถวแนวตั้ง (กี่จุดต่อวินาที) ส่วน Bit Depth คือจำนวนช่องแนวนอน (กี่ระดับความดัง) — ยิ่งกริดละเอียด รูปคลื่นที่ได้ยิ่งเหมือน Analog ต้นฉบับ

ทำไมต้อง 48 kHz สำหรับภาพยนตร์?

มาตรฐาน SMPTE และ EBU กำหนดให้งาน Video Production ใช้ 48 kHz เพื่อให้ไฟล์เสียงซิงก์กับ Frame Rate ของวิดีโอได้อย่างแม่นยำ — กล้อง, NLE Software และ Audio Workstation ทุกตัวรองรับ 48 kHz เป็น Default

เมื่อนำ Sample Rate 44.1 kHz ไปรวมกับไทม์โค้ดวิดีโอ จะเกิด Pull-up/Pull-down ที่อาจทำให้เสียงเหลื่อมกับภาพในระยะยาว

Bit Depth ไม่ได้เพิ่มความถี่สูงเหมือน Sample Rate แต่เพิ่มความละเอียดของ “ความดัง” และเพิ่มพื้นที่ปลอดภัยเวลาบันทึกเสียงเบา/ดังมาก

Bit DepthDynamic Rangeใช้งาน
16-bit~96 dBมาตรฐาน CD Audio

32-bit Floatไม่จำกัดเทคโนโลยีใหม่ในเครื่องอัดภาคสนาม แทบเป็นไปไม่ได้ที่เสียงจะ Clip

ทำไม 32-bit Float ถึงพิเศษ?

32-bit Float ใช้ระบบทศนิยมลอยตัว (Floating Point) แทนที่จะเป็นจำนวนเต็ม (Integer) ทำให้ Dynamic Range แทบไม่จำกัด — แม้เสียงดังมากจนเกิน 0 dBFS ใน Monitor ข้อมูลที่บันทึกไว้ยังคงสมบูรณ์ สามารถนำไป Adjust ใน Post-Production ได้

เครื่องที่รองรับ 32-bit Float เช่น Zoom F6, RØDE Wireless GO II, Saramonic Blink 500 Pro, DJI Mic3

03 · สายสัญญาณ

Balanced vs Unbalanced

Balanced Cable (XLR, TRS)

มี 3 สาย: Hot (+), Cold (−), Ground — ระบบ Differential ยกเลิก Noise ที่เกิดระหว่างทาง (Common Mode Rejection) ทำให้ใช้สายยาว 30-100 เมตรโดยไม่มี Hum เข้า เหมาะกับงาน Professional ทุกประเภท

Unbalanced Cable (TS, RCA)

มีแค่ 2 สาย: Signal และ Ground — ใช้ได้ดีสำหรับสายสั้นไม่เกิน 3-5 เมตร เมื่อสายยาวขึ้น จะเก็บสัญญาณ EMI จากอุปกรณ์ไฟฟ้ารอบข้าง ทำให้เกิด Hum หรือ Buzz

Boom Setup

ชุด Boom มีอะไรบ้าง?

Mic Shotgun
Pattern แคบ (Supercardioid / Hypercardioid) เช่น RØDE NTG3, Sennheiser MKH50 — ตัดเสียงข้างได้ดี
RØDE Blimp
กรงลม (Basket) ครอบ Mic Shotgun — กัน Wind Noise ได้ดีมากสำหรับงานภายนอก
Windshield / Dead Cat
ขนสังเคราะห์คลุม Blimp — ป้องกันลมได้มากกว่า Foam เพียงอย่างเดียว 10-15 dB
Boom Pole
ไม้บูมปรับความยาวได้ เช่น RØDE Boompole — คุณภาพดีไม่ส่งเสียงเสียดสีเมื่อขยับ

หลัก Boom Placement

วาง Mic ให้ใกล้นักแสดงมากที่สุดโดยยังอยู่นอก Frame — ยิ่งใกล้ยิ่งดี เพราะเสียงลดลง 6 dB ทุกครั้งที่ระยะห่างเพิ่มเป็นสองเท่า (Inverse Square Law)

Boom จากบนลงล่างได้ Frequency Response ของ Dialogue ที่ดีกว่าการ Boom จากล่างขึ้นบน เพราะหลีกเลี่ยง Proximity Effect และเสียง Chest ที่มากเกินไป

Stage Monitor & IEM

ระบบ Monitor คือสิ่งที่ Musician ได้ยินบนเวที — แตกต่างจาก PA ที่คนดูได้ยิน

ประเภทของ Monitor

ประเภทคำอธิบายข้อดี
Wedge Monitorลำโพงวางพื้นเอียงหาหูผู้แสดง ต่อผ่าน XLR / Speakonติดตั้งง่าย ทนทาน ราคาประหยัด
IEM (In-Ear Monitor)หูฟัง Wireless ใส่ในหู รับ RF จาก TXลด Stage Volume, Isolation ดีมาก, ได้ยินชัด
Side Fillลำโพงข้างเวทีสำหรับ Drummer / Keysครอบคลุมพื้นที่กว้าง
Drum Fillลำโพงวางหลัง Drum Kitให้กลองได้ยินตัวเองชัด

IEM System — วิธีต่อสาย

Monitor Console ส่ง Aux Mix ผ่าน XLR → IEM Transmitter Rack
IEM Transmitter แปลงสัญญาณ XLR → RF ส่งผ่านอากาศ
Bodypack Receiver รับ RF → 3.5mm TRS → หูฟัง IEM
ต้อง Coordinate Frequency ระหว่าง IEM, Wireless Mic และ RF อื่นๆ บนเวที

Wedge vs IEM — เลือกอะไรดี?

Stage Box & Multicore

ระบบรวม/ส่งสัญญาณจากเวทีไป FOH — ลดความยุ่งเหยิงของสายบนเวทีได้มาก

Stage Box คืออะไร?

Stage Box เป็นกล่องรวมสัญญาณบนเวที รับ XLR จาก Mic/DI Box หลายช่อง แล้วส่งผ่าน Multicore Snake หรือ Digital Network ไปยัง FOH Console — ลดจำนวนสายที่ต้องวิ่งจากเวทีไป FOH เหลือเพียง 1-2 เส้น

Analog Stage Box + Snake — Multicore 16/24/32 ch วิ่งจากเวทีไป FOH (max ~100 เมตร)
Digital Stage Box (Dante/AES50) — ใช้ Cat6 Ethernet แทน Multicore น้ำหนักเบา ระยะไกลขึ้น
Return Lines — ส่ง Monitor Mix กลับจาก FOH/Monitor Console มาที่เวทีผ่าน Snake เดียวกัน

Dante Network — Digital Snake สมัยใหม่

Dante ส่งสัญญาณเสียงผ่าน Cat6 Ethernet รองรับสูงสุด 512 channels ในสาย Cat6 เส้นเดียว Latency ต่ำมาก (~1 ms) ใช้กับ Console รุ่นใหม่เช่น Yamaha CL/QL, Dante-enabled Stage Box

Analog MulticoreDigital (Dante/AES50)
ช่อง16–48 ch ต่อ Snake512 ch ต่อ Cat6 เส้นเดียว
ระยะ~100 เมตร100 ม. (Cat6) / ไกลกว่าผ่าน Switch
น้ำหนักหนัก (สาย Multicore ใหญ่)เบามาก (Cat6 สาย LAN)
Latency0 ms (Analog)~1 ms (ยอมรับได้)
ราคาถูกกว่าแพงกว่า (ต้องการ Network Switch)

Signal Flow

เส้นทางสัญญาณในงาน Live — ตั้งแต่ Mic บนเวทีจนถึง PA Speaker และ Monitor

เส้นทางสัญญาณ FOH (Main PA Path)

Mic / DI Box → XLR → Stage Box Input
Stage Box → Multicore Snake / Cat6 Dante → FOH Console
FOH Console → XLR Main L/R Out → DSP Processor / Crossover
DSP → XLR → Power Amplifier → Speakon → PA Speaker / Sub
Aux Send → Monitor Console → Wedge Monitor (XLR/Speakon) / IEM TX (XLR → RF)

ตาราง Signal Path แบบสรุป

ขั้นตอนอุปกรณ์Connector

StageStage BoxMulticore / Cat6
MixFOH Digital ConsoleDante / AES50 / Analog XLR
ProcessDSP / CrossoverXLR / AES Digital
AmplifyPower AmplifierXLR In / Speakon Out
OutputMain PA / Sub / Line ArraySpeakon
MonitorIEM TX / WedgeXLR → Wedge / RF → IEM

ปัญหาที่พบบ่อยใน Signal Flow

Ground Loop / Hum — เกิดจาก Unbalanced Cable ยาว หรือ Ground ต่างระดับ → ใช้ DI Box หรือ Ground Lift
Feedback — Mic รับเสียงจาก Speaker กลับเข้ามา → ลด Gain, ใช้ EQ ตัด, จัด Mic Placement
RF Dropout — IEM / Wireless Mic หาย → ตรวจ Frequency Coordination และ Antenna Placement
Gain Structure ผิด — Clip ที่ Preamp ทำให้เสียงแตกตลอด Chain → ตั้ง Gain ที่ -18 dBFS สำหรับ Average
